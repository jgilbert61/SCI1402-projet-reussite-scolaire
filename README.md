# Projet SCI1402 – Réussite scolaire

## Description
Ce projet vise à analyser les facteurs associés à la réussite scolaire à partir
d’un jeu de données provenant de la plateforme Kaggle.

## Jeu de données
- Source : Kaggle – Students Performance
- Nombre d’observations : 1000 élèves
- Variables principales : notes, genre, type de lunch, préparation au test, etc.

## Méthodologie
- Nettoyage et préparation des données
- Analyse exploratoire
- Création de variables (moyenne, réussite/échec)
- Modélisation (régression linéaire)

## Outils utilisés
- R
- RStudio
- R Markdown
